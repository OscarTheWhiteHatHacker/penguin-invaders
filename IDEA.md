# Penguin Invaders — Arcade Shooter

Defend your penguin by defeating polar bears in a Space Invaders-style arcade shooter.

## Features
- **7 wave patterns** — THE WALL, V FORMATION, DIAMOND, PINCER, SWARM, TANKS, ARROWHEAD
- **3 difficulty levels** — Easy, Medium, Hard
- **Power-ups** — Spread Shot, Rapid Fire, Shield, 1UP, Nuke (drop from killed enemies)
- **Boss waves** every 5 waves — giant bear with crown, red eyes, HP bars, special SFX
- **Combo system** — consecutive kills build multiplier up to 4x score
- **High score** — persisted via localStorage with celebration on new records
- **Pause** — P key with blue glowing overlay
- **Score popups** — floating +points on kills
- **Wave announcements** — animated banner on each new wave
- **Arcade chiptune** — Web Audio API melody + bass + drums
- **Screen shake, particles, snowflakes, twinkling stars**
- **Fullscreen support**
